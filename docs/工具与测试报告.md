| **OpenSpec**（间接使用） | Spec-Superflow 底层依赖，提供 `specs/` 增量 + spec-merger | 规格合并 |
| **pytest 9.1.1** | 自动化测试 | 测试框架 |
| **pytest-cov 7.1.0** | 覆盖率统计 | 测试工具 |
| **pip + venv** | 包管理 + 隔离环境 | 工程化 |
